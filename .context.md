# SOC Job Task Analyzer - AI Assistant Context

## Project Overview

**Name:** SOC Job Task Analyzer
**Version:** 1.0.0 (Production Ready)
**Domain:** Cybersecurity Workforce Research & Data Analysis
**Purpose:** Automated pipeline for collecting, classifying, and analyzing SOC job descriptions using advanced data processing techniques

## Core Capabilities

### Pipeline Stages
1. **Data Collection** - SerpAPI integration for Google Jobs data
2. **Classification** - Rule-based SOC Tier 1 categorization
3. **Deduplication** - Fuzzy string matching (72.5% reduction achieved)
4. **Thematic Clustering** - Keyword-based grouping into 10 SOC themes (82.8% coverage)
5. **Analysis & Reporting** - Multiple output formats with comprehensive metadata

### Performance Metrics
- **Execution Time:** <14 seconds for complete pipeline
- **Data Processing:** 1000+ jobs with memory-efficient handling
- **Accuracy:** Research-validated results with quality assurance
- **Scalability:** Parallel processing and optimization capabilities

## Technical Architecture

### Core Modules
- `src/job_run.py` - Pipeline orchestration and main entry point
- `src/soc_scrapper_API.py` - SerpAPI integration with pagination
- `src/data_analyzer.py` - SOC classification engine
- `src/task_aggregator.py` - Fuzzy deduplication algorithms
- `src/task_thematic_clusterer.py` - Thematic clustering logic

### Configuration System
- `configs/rules.json` - Classification rules and criteria
- Environment variables for API keys and settings
- JSON-based pipeline configuration
- Modular design for easy customization

### Data Flow
```
Raw Jobs (JSON) → Classification → Deduplication → Clustering → Outputs (JSON/CSV)
     ↓              ↓              ↓              ↓              ↓
  SerpAPI       SOC Rules     Similarity      Keywords      Reports
  Collection    (Tier 1)      Matching        Matching       Analysis
```
