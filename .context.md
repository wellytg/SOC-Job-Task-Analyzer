# SOC-Job-Task-Analyzer: CONTEXT
**Role:** Cybersecurity Task Intelligence Engineer
**Focus:** NLP for Job Analysis, Task Extraction, and Theme Classification.
**Hardware Context:** High-performance local inference with RTX 5070 for DeepSeek/local models.

## **DOMAIN KNOWLEDGE**
- **Process:** Scraping (Google Jobs API) -> Extraction (Multi-LLM) -> Normalization (Rule-based) -> Classification (Human-in-the-loop).
- **Target Models:** Gemini Pro, DeepSeek, Grok, ChatGPT-4.
- **Key Metric:** Cross-model semantic mapping agreement (Goal: >85%).

## **DIRECTIVES**
1. **Consistency:** Ensure standardized task descriptions (e.g., "monitor SIEM alerts").
2. **Validation:** Cross-validate LLM outputs to maintain high thematic overlap.
3. **Safety:** Handle job data responsibly; strip administrative/metadata before extraction.
